# Tic Tac Toe 🎮
A responsive Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features
- Two-player gameplay
- Winner detection
- Draw detection
- Responsive design
- Score tracking
- Winning block highlight
- Reset and New Game buttons

## Tech Stack
- HTML
- CSS
- JavaScript

## What I Learned
- DOM manipulation
- Event listeners
- Game logic implementation
- Responsive web design
- CSS Flexbox
- Dynamic class handling

## Future Improvements
- AI opponent
- Sound effects
- Animations
- Online multiplayer

## How to Run
1. Clone the repository
2. Open `index.html` in browser
